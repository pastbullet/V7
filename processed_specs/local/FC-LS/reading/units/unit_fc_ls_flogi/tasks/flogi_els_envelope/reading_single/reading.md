In section 4.1, the specification defines the generic Extended Link Service envelope used to identify FLOGI traffic. An ELS request or reply is a single Sequence, and the ELS_Command code is carried in the first word of the Payload of the first frame of that Sequence. ELS request and corresponding reply are performed within a single Exchange. The TYPE field for ELS frames is set to 01h. Table 3 identifies FLOGI as the ELS command with value 04h, described as F_Port Login and abbreviated FLOGI.
