FC-LS section 4 defines the generic ELS envelope used to identify FLOGI. An ELS request or reply is a single Sequence, with the ELS_Command code carried in the first word of the Payload of the first frame; the first byte of that Payload is the ELS_Command code and later frames in the Sequence do not repeat it. Table 3 identifies FLOGI (F_Port Login) with ELS_Command value 04h. ELS frames use TYPE 01h. R_CTL is set per Table 2, where routing 0010b with information 0010b denotes Request and routing 0010b with information 0011b denotes Reply. The request and corresponding reply are required to occur within a single Exchange.
