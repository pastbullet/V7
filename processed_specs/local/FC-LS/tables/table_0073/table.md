| Value | Meaning |
| --- | --- |
| 00h | The Time Stamp field is unknown or unspecified. |
| 01h | Time Server: The 64 bit time stamp is reported in units of seconds and<br>fractions of a second. The time stamp uses the value obtained from the Time<br>Server (see FC-GS-4). |
| 02h | Clock synchronization format: The 64-bit time stamp is reported as defined for<br>the Clock Synchronization Update (CSU) ELS. |
| 03h to FFh | reserved |
